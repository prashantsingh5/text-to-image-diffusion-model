# Code of Conduct

## Our Pledge

We are committed to making participation in this project a respectful and harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Using welcoming and inclusive language
- Respecting differing viewpoints and experiences
- Giving and receiving constructive feedback
- Focusing on what is best for the community and the project

Examples of unacceptable behavior:

- Harassment, discrimination, or personal attacks
- Trolling, insulting, or derogatory comments
- Publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct and may remove comments, commits, issues, or pull requests that violate these standards.

## Scope

This code of conduct applies within all project spaces, including issues, pull requests, and discussions.
